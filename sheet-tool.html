🕐 Tao code Random Timestamp

Set thong so → Lay code → Dan vao Apps Script trong Google Sheet → Chay.

VD: 7 → 23 = chi random tu 7h sang den 11h dem (bo 0h-6h)

📖 Huong dan

  1. Mo Google Sheet can sua
  2. Tien ich mo rong → Apps Script
  3. Xoa het code cu, dan code tren vao
  4. Ctrl+S luu → Bam ▶ Chay
  5. Cho phep quyen → Xong!

📋 Copy Sheet (Xoa lich su)

Tao ban sao spreadsheet — sheet moi se sach lich su chinh sua, khong co to xanh.

📖 Huong dan

  1. Mo Google Sheet goc (cai can copy)
  2. Tien ich mo rong → Apps Script
  3. Dan code → Ctrl+S → ▶ Chay
  4. File moi se xuat hien trong Drive, sach lich su

📅 Set Ngay Sua File

Thay doi ngay sua cuoi cung cua file Google Drive.
Luu y: Ngay tao (createdTime) khong sua duoc, chi sua duoc ngay chinh sua (modifiedTime).

📖 Huong dan

  1. Mo file can doi ngay trong Google Drive
  2. Tien ich mo rong → Apps Script
  3. Dan code → Ctrl+S → ▶ Chay
  4. Kiem tra lai ngay sua trong Drive
